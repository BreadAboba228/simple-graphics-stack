A simple graphics stack written from scratch. So far, it uses only the CPU.

![cube_example](./crates/simple-3d/docs/cube_example.gif)

### [Simple-linear-alrebra](./crates/simple-linear-algebra)
A library that implements vectors, matrices, and quaternions.
### [Simple-render](./crates/simple-render)
A library that implements rendering via minifb. Double buffering is used.
### [Simple-3d](./crates/simple-3d)
A library that implements 3D. There is work with scenes, camera and perspective projection.
